AI 学习之路

学习资料

  本篇 blog 主要想分享一下我在人工智能学习中推荐的一些课程或是书籍，课程和书籍都是我看完觉得很好的，谨在此做分享：

编程语言

  python 是我上大学以来学的第一个编程语言，初步入门是在 B 站上黑马程序员的课程（《黑马程序员Python小白基础入门教程 Python入门到精通教程》 )，后面遇到不会的就是自己搜集一些博客来看一看了。
  c 语言 和 Java 非常推荐翁恺老师的 《c 语言程序设计》 和 《面向对象程序设计——Java语言》。
  至此，了解完编程语言的逻辑后，在 c++ 的学习中就没有看课，大致浏览了一下细节语法就开始了刷题模式，在题目中去学习和练习，平常练习的话就在 leetcode 以及 洛谷 上。（本人曾经立志于成为一名 ACMer，但是转专业后课程实在太多，所以放弃掉了，想参加比赛的可以去atcoder 和 codeforces 做比赛)。

数学基础

  我在大一学习完高数之后，转来人工智能又去补修了数学分析以及高等代数，但是木有学过线代（痛！）。高数的话我觉得大部分考研老师讲的都很好，找一个自己喜欢的就好嘞。数分和高代推荐复旦大学陈纪修老师的《数学分析》以及谢启鸿老师的《高等代数学》，讲的都超级好，很细致，给了初来乍到的鼠鼠努力学习的希望。

408 学习

  由于学校没有开汇编以及数电的课，所以我的 408 学习就单纯的针对 408 统考的四门课程，数据结构当时学的很认真，去刷了青岛大学王卓老师的课程 《数据结构》，计算机组成原理看了哈工大刘宏伟老师的 《计算机组成原理》，四门课程又都在期末复习的时候看了看王道的 408 课程（咸鱼老师讲的还是非常好的！）。

人工智能专业课

  作为大三的本科生，专业方面开始学习的时间也还不到一年（大一不是本专业的，所以我是不是应该算大二的！），没有太理清专业方向的学习内容，单纯把自己学习的课程和书籍罗列出来以供参考。

机器学习：

  • 李航老师的《统计学习方法》
  • 浙江大学胡浩基老师的《机器学习与人工智能》

深度学习：

  • 首当其冲的是河北工业大学刘洪普老师的《PyTorch深度学习实践》
  • 斯坦福大学的《cs231n》课程
  • 炮哥的《Pytorch框架与经典卷积神经网络与实战》,
  • 李沐老师的《动手学深度学习 PyTorch版》(沐神的论文精读合集做的也特别好，非常推荐！)
  • 吴恩达老师的《生成对抗网络》

网站：

  • github
  • dblp
  • paperswithcode
  • autodl

  了解完底层逻辑后，我觉得更进一步的学习是去浏览经典文献以及最新文献，大部分经典文献都能在网上找到详细的解读，但我觉得更重要的还是自己去阅读，去思考，去复现，所以未来的路更多的需要靠自己去完成。

  暂时就想到这些，后续肯定会再补充的，鼠鼠的 AI 学习之路才刚刚开始！
